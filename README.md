# SKThanos
Thanos gauntlet in skript

Needed
- Skript (Link: https://github.com/SkriptLang/Skript/releases)
- SKBee (Link: https://github.com/ShaneBeee/SkBee/releases/tag/1.12.4)
- SkQuery (Link: https://github.com/SkQuery/SkQuery/releases/tag/4.1.5)


How to use
1) Download the folder
2) Put the "ThanosPack.Zip" into the resource pack folder
3) Put the "Thanos" folder in your skripts folder
4) once the folder is in your skript folder so "/sk reload all"
5) then do "/stone all" and click the gaunlet
